# Birat Chapagain

> token-maxxing in the ruins of the attention economy

I build small machines that make large machines less confused.
Mostly AI tooling, developer workflows, knowledge systems, and whatever else helps humans survive the autocomplete era with a little dignity left.

Currently experimenting with:

- context engineering for agents
- local-first knowledge formats
- tools that make LLMs cite their sources instead of hallucinating with confidence
- turning chaos into Markdown, JSON, and occasionally a working CLI

## Operating Mode

```text
input:  vague idea + too many tabs open
process: ship prototype, interrogate model, compress context, repeat
output:  one more repo the machines can read
```

I believe the future is not AGI replacing everyone.
It is probably worse: everyone manually fixing YAML generated by five different agents while a dashboard says productivity is up 300%.

So I am building tools for that world.
